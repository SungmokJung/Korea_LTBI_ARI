# Estimation of annual risk of Mycobacterium tuberculosis infection

This repository contains the underlying code for
(1) describing the dataset and estimating adjusted prevalence and positive predictive values, and
(2) estimating the annual risk of infection and age-specific prevalence of latent tuberculosis infection (LTBI) in South Korea.

## Note
This repository is under active development and will be continuously updated.
